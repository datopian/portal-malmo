Inom Malmö produceras stora mängder data varje dag, och vi lagrar mycket historisk data. Öppna data handlar om att dela den data vi har med medborgare, forskare, media och företag, bland annat.

Vi har inte alltid svaren på hur data kan användas eller vem som kan behöva den. Genom att arbeta med öppna data skapar vi möjligheter för andra att använda den data som finns.
